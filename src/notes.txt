La Class Circuit :

 - S'occupe de stocker tout le graphe, donc un vector de IComponents.
 - Mettre les methods pour manipuler le graphe entier

La Class IComponents :

 - Représente l'interface d'un component globallement
 - aucune valeur n'est mise, juste des methods pour les manipuler

La Class AComponents :

 - Hérite de IComponents, peut override si on veut les methods
 - Stock un vector pour les input, et un pour les outputs.

Les Class components :

 - Hérite de AComponents et IComponents
 - C'est individuellement chaque opération que l'on peut faire
